
# Portfolio Website

A portfolio website build using create react app.

- To display my projects,
- Skills and services
- Resume 
- Contact and other info
## Run Locally

Clone the project

```bash
  git clone https://github.com/Vyshnav-KS/portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
    yarn install
```

Start the server

```bash
  yarn start
```
